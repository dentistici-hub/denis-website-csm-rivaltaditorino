# C.S.M. SNC di Bertacco F. & C.

**Live preview:** [https://dentistici-hub.github.io/denis-website-csm-rivaltaditorino/](https://dentistici-hub.github.io/denis-website-csm-rivaltaditorino/)

Static marketing site for C.S.M. SNC di Bertacco F. & C. — carpenteria metallica storica di Rivalta di Torino, oltre 50 anni di lavorazione del ferro e dell'acciaio inox per privati, aziende ed enti pubblici. Built with Astro and deployed to GitHub Pages as a demo preview.

## About this site

- **Client:** C.S.M. SNC di Bertacco F. & C. (P.IVA 04896520017)
- **Industry:** Construction / metal fabrication (carpenteria metallica)
- **Location:** Via Orbassano 48, 10040 Rivalta di Torino (TO), Italy
- **Source site (for reference):** https://www.csm-rivaltaditorino.it/

## Build

```bash
npm install
npm run dev      # local preview at http://localhost:<port>/denis-website-csm-rivaltaditorino/
npm run build    # static output in dist/
npm run preview  # serve built output
```

## Deploy

Automated via `.github/workflows/deploy.yml` on push to `main`. GitHub Pages serves from the `main` branch via the workflow artifact.
